# FavLoc
A basic app with the ability to pin your favorite places on google maps 

# Read this before installing the app

This app heavily relies on Places API and google maps if you want to gain the fully experience of this app you need to provide API Key for Google Maps under
[`strings.xml`](https://github.com/subhipandey/FavLoc/blob/master/app/src/main/res/values/strings.xml).

# How to get the API Key

Go to your google cloud console there you will a section named 'API & Services' under Credentials section you a generate a new api key.
